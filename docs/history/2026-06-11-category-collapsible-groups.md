# 작업 이력: 카테고리 설정에 대분류 접이식 그룹

- **날짜**: 2026-06-11
- **작업자**: 사용자
- **브랜치**: main

## 변경 요약

설정 페이지 카테고리 탭의 평면 테이블을 (구분, 대분류) 단위 접이식 그룹으로 개편. 그룹 헤더 행(셰브론+대분류명+구분 배지+소분류 개수)을 클릭해 펼침/접힘을 토글하며, 기본은 전체 펼침이고 CRUD 후에도 접힘 상태가 유지된다.

## 변경 파일 목록

- `frontend/src/pages/SettingsPage.tsx` - `CategoriesTab`에 그룹핑·접힘 상태(Set)·그룹 헤더 행 추가, 기존 `·` 의사 그룹 표시 제거

## 상세 변경 내용

상세: [docs/tasks/2026-06-11-category-collapsible-groups](../tasks/2026-06-11-category-collapsible-groups/) 참조

## 테스트 방법

- `npm run build`, `npm run lint` 통과
- 수동: 설정 → 카테고리 탭에서 그룹 헤더 클릭 토글, 카테고리 추가/수정/삭제 후 접힘 상태 유지 확인
